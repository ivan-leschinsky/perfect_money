# PerfectMoney

Gem api for https://perfectmoney.is/ into your ruby project

## Installation

Add this line to your application's Gemfile:

    gem 'perfect_money'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install perfect_money
